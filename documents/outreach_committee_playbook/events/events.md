## How the Outreach Committee proposes events to the Governing Board

The Outreach Committee proposes a prioritized list of events to the Governing Board. 

The Outreach Committee Event SIG is responsible for proposing events to the Governing Board. All proposed events must come from the events board.

### Responsibility

1. Outreach Committee Events SIG is responsible for collecting and prioritizing events of interest to NeoNephos.
2. Outreach Committee Chairperson is responsible for proposing these events to the Governing Board.

### Procedure

1. Outreach Committee Events SIG ensures for each issue:
    1. All form entries of the ```Add Event``` template are filled out to the satisfaction of the Outreach Committee Events SIG.
    2. The engagement type label ```booth``` and / or ```talk``` is given.
2. Outreach Committee Chairperson is responsible for creating the events package for consideration of the board:
    1. Go to the [Governing Board Repository page](https://github.com/neonephos/gb/issues).
    2. Fill out the ```Propose Events to the board``` issue template.
    3. Link the events of the [events board](https://github.com/orgs/neonephos/projects/8/views/1) to the newly created issue through ```Add existing issue```.
    4. Ensure that the Governing Board Chairperson is assigned to this request
